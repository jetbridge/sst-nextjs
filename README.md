# SST-NextJS

## What is this?

A starter kit for NextJS + Serverless Stack.

Includes:

- NextjsSite construct
- Eslint
- Prettier
- Husky
- Shared core library
- NextJS `app` dir
- GitHub action
- pnpm

## Quickstart

Install [Node](https://nodejs.org/en/download/).

### Install dependencies

```shell
npm i -g pnpm
pnpm i
```

### Run backend dev server

```shell
pnpm dev
```

### Run frontend dev server

```shell
pnpm dev:web
```
